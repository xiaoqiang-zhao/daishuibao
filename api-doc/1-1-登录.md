# 登录接口

## url

```
POST: /api/login
```

## 入参

`userName`  用户名
`password`  密码

## 出参

### 登录成功

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": true,
        "userName": "asd8664sdnhjsfbblljssskuppqzk"
    }
}
```

`data.isSuccess` 是否登录成功
`data.userName` 登陆后的用户名，为将来扩展预留，可暂时不实现，保证唯一

### 登录失败

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": false
    }
}
```
