# 添加公司

> 用于客户档案页面

## url

```
PUT: /api/company/:companyId/suppliers-and-customers
```

## 入参

```json
{
    "suppliers": [
        "AAA 有限公司"
    ],
    "customers": [
        "BBB 公司"
    ]
}
```

`supplier` 供应商公司名称列表

`customer` 客户公司名称列表，

## 出参

```json
{
    "code": 0,
    "message": "",
    "data": {
        "suppliers": [
            "AAA 有限公司"
        ],
        "customers": [
            "BBB 公司"
        ]
    }
}
```

`suppliers` 供应商公司名称列表

`customers` 客户公司名称列表
