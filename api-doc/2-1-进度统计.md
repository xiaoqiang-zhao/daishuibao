# 进度统计接口

> 用于客户档案、信息上传、智能做账、一键报税 等模块的一级页面中。

## url

```
GET: /api/processStatistics
```

## 入参

无

## 出参

```json
{
    "code": 0,
    "message": "",
    "data": {
        "total": 3175,
        "pageStatusList": [
            {
                "pageStatus": 1,
                "amount": 1322
            },
            {
                "pageStatus": 2,
                "amount": 1091
            },
            {
                "pageStatus": 3,
                "amount": 762
            }
        ],
        "bizStatusList": [
            {
                "status": 1,
                "amount": 762
            },
            {
                "status": 2,
                "amount": 19
            }
        ]
    }
}
```

状态附录:

pageStatus:
- 1: 信息上传环节
- 2: 智能做账环节
- 3: 一键报税环节

biz status
- 1: 财务完成
- 2: 报税完成
